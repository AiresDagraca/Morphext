Morphing + Text = Morphext

The simplest text rotator powered by jQuery and Animate.css

Download Source (GitHub)

Installation

Via Bower:
bower install Morphext --save
Import CSS:
<link rel="stylesheet" href="bower_components/Morphext/dist/morphext.css">
Import JS:
<script src="bower_components/Morphext/dist/morphext.min.js"><script>

Usage

HTML:
I am a <span id="js-rotating">So Simple, Very Doge, Much Wow, Such Cool</span> Text Rotator
Initiate (JS):
$("#js-rotating").Morphext({
    // Animation type (refer to Animate.css for a list of available animations)
    animation: "bounceIn",
    // An array of words to rotate are created based on this separator. Change it if you wish to separate the words differently (e.g. So Simple | Very Doge | Much Wow | Such Cool)
    separator: ",",
    // The delay between each word in milliseconds
    speed: 2000
});
